## 📌 Descripción
Proyecto de análisis espacial y predictivo que identifica las zonas 
de mayor potencial comercial para apertura de cafeterías en las 16 
alcaldías de la CDMX, combinando datos socioeconómicos del INEGI, 
datos de competencia del DENUE y análisis geoespacial.

## 🎯 Pregunta de investigación
**¿Qué factores socioeconómicos y espaciales determinan el éxito 
comercial de una cafetería en la CDMX, y cuáles son las zonas 
de mayor oportunidad?**

## 📊 Metodología
1. 📥 Recolección de datos — INEGI Censo 2020 + DENUE 2024
2. 🧹 Limpieza y preparación de datos
3. 📊 Análisis exploratorio — estadística descriptiva
4. 🗺️ Análisis espacial — Índice de Moran, mapas de calor
5. 🤖 Modelos predictivos — OLS + Random Forest
6. 🏆 Sistema de scoring por alcaldía
7. 📈 Dashboard ejecutivo en Power BI
